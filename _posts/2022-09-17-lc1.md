## 第一题：数组序号转换 1331

### Solution
```c++
class Solution {
public:
    vector<int> arrayRankTransform(vector<int>& arr) {
        vector<int> b = arr;
        sort(b.begin(), b.end());
        b.erase(unique(b.begin(), b.end()), b.end());
        for(auto &x: arr) {
            x = upper_bound(b.begin(), b.end(), x) - b.begin();
        }
        return arr;
    }
};
```

### 思路
既然输出的是原数组的
